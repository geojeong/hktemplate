# hktemplate
정상혁 보고서 양식

Files:

- hktem.sty: style file
- main.tex: sample main file

How to compile:

- Use XeLaTeX or LuaLaTeX
- Locally: `xelatex main.tex`

Notes:

- Main font is NanumMyeongjo
- Template is inspired by CVPR layout but simplified for coursework/reports
- If Nanum fonts are missing on your local machine, install NanumMyeongjo and NanumGothic
